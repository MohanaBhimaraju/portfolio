# Mohana Kumari Bhimaraju - Portfolio

A high-end, immersive, anti-gravity themed portfolio website showcasing a dual career as a **Full Stack Java Developer** and a **Freelance Model**.

## 🚀 Features
- **Futuristic UI**: Deep-space cosmic theme with glassmorphism and neon aesthetics.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Dual Portfolio**: Showcases both technical projects and freelance modeling work.
- **Smooth Animations**: Intersection Observer based reveal animations and floating effects.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom variables, glassmorphism, and complex animations.
- **JavaScript**: Interactive navigation and scroll animations.
- **Google Fonts**: Orbitron, Rajdhani, and Inter for a modern look.
- **FontAwesome**: High-quality icons for skills and social links.

## 📁 Project Structure
- `index.html`: Main entry point.
- `css/`: Stylesheets.
- `js/`: Script files.
- `images/`: High-quality project and portfolio assets.
